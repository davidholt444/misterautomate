LinkedIn Feed Filter is a browser extension project built to create a more intentional feed experience by bringing the most relevant people and conversations closer to the surface.

A smaller, direct build with focused scope, practical use case, and a clear download path for Chrome and Edge.

Project summary

What it does

The extension is designed to reduce feed noise and make LinkedIn feel more aligned with the people and posts you actually care about seeing.

  • Brings relevant connections forward
  • De-emphasizes suggested and promoted clutter
  • Runs as a browser-based local install
  • Maintains a simple direct-use delivery model

Why it belongs here

This project is smaller in scope than Compass, but it reflects the same instinct: remove friction, reduce noise, and build something concrete around a real user experience problem.

Installation

  1. Download and unzip the file above.
  2. Open Chrome or Edge and go to chrome://extensions or edge://extensions.
  3. Enable Developer mode.
  4. Choose Load unpacked and select the unzipped folder.
  5. Open LinkedIn and the extension will activate automatically.
